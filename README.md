# ibus-daemon-de

SystemD user service for IBus (desktop environment specific).

## Description

This is source code and PKGBUILD for [ibus-daemon-de][ibus] AUR package.

## Installation

```sh
yay -S ibus-daemon-de
```

## Usage

Enable

```sh
enable-ibus-daemon
```

Disable

```sh
enable-ibus-daemon -d
```
